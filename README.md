## Install the extension:

1. Install the extension
2. Create a folder named `warm-beige-theme` and place the files inside it
3. Open VS Code
4. Press `Ctrl+Shift+P` (Windows/Linux) or `Cmd+Shift+P` (Mac)
5. Type: `Developer: Install Extension from Location`
6. Select the folder

## Activate the theme:

Press `Ctrl+K Ctrl+T` → select **"Warm Beige"**
